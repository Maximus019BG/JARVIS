# JARVIS end goal

JARVIS build on raspberry pi 5 with imx 500 cam is smart workstation assistant. Its job is to optimize human productivity at the workstation while having a cloud platform.

### JARVIS contains:
    - Voice navigation
    - Hand detection
    - Voice responding 
    - Visual responding
### Features:
    - Coding 
        - Building apps
        - Running code
        - Optimizing code
    - Blueprint creation
        - Buildings
        - Small elements (overlay)
        - Real world scale 
    - Every day questions / tasks
    - Web search
    - External tools
    - Connection to other devices
        - Working with files from them
        - Working with apps on them
    - Cloud platform for file sharing and more
    - Automation
        - N8n look a like
        - automations with jarvis hardware 
        - automations with external tools
    - Fully customizable UI
    - JARVIS as a MCP server
    - Multi jarvis link 
        - More resources 
        - Faster
        - Better results
    - Secure communication
        - HMAC
        - Replay Protection
            - X-timestamp
            - Nonce
        - Communication with https (other protocol could work too)
